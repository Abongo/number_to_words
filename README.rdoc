= Number To Words

Rails gem/plugin to describe a number in words (Spanish).

== Install

Add the following line to your Gemfile:

  gem "number_to_words"

== Usage

For now it only works for spanish:

  1235.to_words
  => "un mil doscientos treinta y cinco"
